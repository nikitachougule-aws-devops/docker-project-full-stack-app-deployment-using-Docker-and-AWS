# Full Stack App Deployment using Docker + AWS

## Project Overview
This project demonstrates how to build, containerize, and deploy a Node.js application with MongoDB using Docker and AWS EC2.

---

## Architecture
User → Backend (Node.js) → Database (MongoDB)  
             ↓  
          Docker  
             ↓  
            AWS EC2  
            
---

## Tech Stack
- Node.js  
- MongoDB  
- Docker  
- Docker Compose  
- AWS EC2  

---

## Features
- Add tasks  
- View tasks  
- Persistent storage using MongoDB  
- Containerized application using Docker  
- Multi-container setup using Docker Compose  
- Cloud deployment on AWS
